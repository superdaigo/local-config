#+TITLE: Ghostty

Fast and feature rich terminal emulator supporting multiple environments.
https://ghostty.org/

* Requirement
- Ghostty (https://ghostty.org/)
- HackGen Console NF (https://github.com/yuru7/HackGen)

* Generate config file
#+begin_src elisp :results none
  ;; Generate file(s)
  (org-babel-tangle-file "ghostty.org")
  (setq-local config-files (list "config"))
  (setq-local tmp-dir "ghostty")
  (setq-local dst-dir (expand-file-name "~/.config/ghostty"))
  ;; Copy current files and deploy
  (setq-local backup-prefix (format-time-string "%Y-%m-%d_%H%M%S_"))
  (dolist (config-file config-files)
    (setq-local tmp-file (file-name-concat tmp-dir config-file))
    (setq-local bak-file (file-name-concat tmp-dir (concat backup-prefix config-file)))
    (setq-local dst-file (file-name-concat dst-dir config-file))
    ;; Take backup
    (if (file-exists-p dst-file)
        (copy-file dst-file bak-file)
      nil)
    ;; Save new file
    (copy-file tmp-file dst-file t))
#+end_src

* Config
See the official document for the details
https://ghostty.org/docs

Need to install the font "HackGen Console NF". Donwload the file from Github repository.
https://github.com/yuru7/HackGen

#+begin_src conf :tangle ghostty/config :mkdirp yes
  # -*- mode: conf; encoding: utf-8; -*-
  # Automatically generated by ghostty.org file

  # Font
  font-family = "HackGen Console NF"
  font-size = 16
  # Background
  background-opacity = 0.9
  # No initial window
  initial-window = false
  # Use non-native full screen
  macos-non-native-fullscreen = true
  # Yakuake-like termial
  quick-terminal-size = 60%
  quick-terminal-animation-duration = 0
#+end_src

** Keybindings
| Action              | Linux              | macOS             |
|---------------------+--------------------+-------------------|
| Reload config       | ~Ctrl + Shift + ,~ | ~Cmd + Shift + ,~ |
| Launch quick window | ~Ctrl + `~         | ~Cmd + Escape~    |

Keybinds are different per OS

Mac keybinding
#+begin_src conf :tangle (if (eq system-type 'darwin) "ghostty/config" "")
  # Yakuake-like termial
  keybind = global:cmd+escape=toggle_quick_terminal
#+end_src

Linux keybinding
#+begin_src conf :tangle (if (eq system-type 'gnu-linux) "ghostty/config" "")
  # Yakuake-like termial
  keybind = global:ctrl+backquote=toggle_quick_terminal
#+end_src
